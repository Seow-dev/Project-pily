## <b> 특징 </b>

- ### <b> 소셜로그인을 이용한 닉네임만 저장하여 유저관리 </b>
- ### <b> 일상을 별점 및 카카오 맵을 이용하여 피드로 기록 </b>
- ### <b> 매거진에 대한 좋아요 기능과 해당 유저 구독 기능 </b>
- ### <b> 페이지네이션 기법을 기반으로 검색결과 출력 </b>
